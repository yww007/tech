# 技术文档网站

一个自动更新的技术文档网站，每天自动更新最新的技术内容。

## 功能特点

- 📅 每天北京时间早上9点自动更新
- 📚 每天北京时间10点自动归档到历史
- 🌙 支持深色模式
- 📱 响应式设计，支持移动设备

## 网站结构

```
tech-blog/
├── index.html          # 首页
├── history.html        # 历史页面
├── about.html          # 关于页面
├── contact.html        # 联系页面
└── history/            # 历史存档目录
    └── 2026/
        ├── index.html  # 年份索引
        └── 03/
            ├── index.html  # 月份索引
            └── 20260329.html  # 日期页面
```

## 自动更新

网站每天北京时间早上9点自动更新，10点自动归档到历史页面。

## 技术栈

- HTML5
- CSS3
- JavaScript
- GitHub Pages
